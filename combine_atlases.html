Atlas Combiner

Merge per-animation sprite sheets into one wide atlas PNG that the brawler engine can load.
How it works: each input sheet must be 4 columns × 8 rows (4 frames × 8 facings). Drop one sheet into each anim slot. The tool stacks them horizontally in the order shown and exports a single PNG of size (cols × 256) × (8 × 256). Drop the result into away_game_brawler_assets/character_atlases/ using the suggested filename, then refresh the game.
